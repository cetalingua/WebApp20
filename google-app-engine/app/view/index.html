MANATEE CHAT DEMO ML MODEL

We are happy to present our first deployed deep learning model that can identify files with manatee calls and mastication sounds.

This model was trained on a small but labeled dataset (see our Zooniverse project) and currently has 95% accuracy rate.

You can check model performance by selecting random spectrogram from Manatee Chat data that were not used for model training or validation (“Select Random Image”) or by using spectrogram URL/uploading the spectrogram.
